Using the following table and the list in [cost.md](cost.md), associate the specified {animal} with the destruction of the corresponding {item}, which is has the corresponding {cost}.

| animal | item |
| ----- | ---- |
| cat | curtains |
| dog | slippers |
| chimpanzee | house |
| gorilla | city |

If none of these animals are specified, use "car" as the item.
